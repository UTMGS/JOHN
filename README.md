# ジョン（JOHN）

ジョン（JOHN）は、将棋やシミュレーションRPGの要素を取り入れた、トランプを用いる2人対戦型のボードゲームです。  
4×4 のコンパクトな盤面で、短時間でも戦略的な駆け引きを楽しめます。

## 特徴

- 4×4 の小さな盤面での濃密な戦術性
- トランプのランダム性と読み合いが融合した戦略性
- 小学生から大人まで、誰でもプレイできる設計
