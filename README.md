# ジョン【トランプボードゲーム】

「**ジョン**」は、将棋やSRPGの要素を取り入れた、2人対戦の戦略トランプボードゲームです。

##  特徴

- 4×4マスのシンプルな盤面で奥深い駆け引き
- トランプを使って戦略・心理・運を楽しむ
- 小学生から大人まで楽しめる
